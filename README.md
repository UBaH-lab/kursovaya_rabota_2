# Курсовая работа: Получение и обработка данных о самолетах через API

## Описание проекта

Проект представляет собой консольное Python-приложение для получения, обработки, фильтрации и сохранения информации о самолетах, находящихся в воздушном пространстве выбранной страны.

Данные получаются с использованием внешних API:
- **Nominatim OpenStreetMap API** — для определения географических границ страны
- **OpenSky Network API** — для получения данных о самолетах в заданной области

Программа реализована с применением:
- объектно-ориентированного программирования (ООП),
- абстрактных базовых классов (ABC),
- работы с внешними API через библиотеку `requests`,
- сохранения данных в JSON-файл,
- фильтрации, сортировки и отображения данных,
- модульного тестирования с `pytest`,
- анализа покрытия тестами с `pytest-cov`.

---

## Функциональные возможности

Программа позволяет:

- ввести название страны;
- получить список самолетов в воздушном пространстве этой страны;
- сохранить данные о самолетах в JSON-файл;
- показать топ N самолетов по высоте;
- показать топ N самолетов по скорости;
- отфильтровать самолеты по стране регистрации;
- отфильтровать самолеты по диапазону высот;
- показать только самолеты в воздухе;
- показать только самолеты в воздухе с высотой больше 0;
- вывести список самолетов в консоль.

1. Клонирование или распаковка проекта

2. Создание виртуального окружения
```bash

```



Особенности реализации
1. Абстрактный класс API
Файл api_base.py содержит базовый абстрактный класс для взаимодействия с API.

2. Абстрактный класс хранилища
Файл file_base.py содержит базовый интерфейс для классов сохранения данных.

3. Класс Aeroplane
Реализует:

валидацию данных через свойства,
преобразование объекта в словарь,
создание объекта из словаря,
перегрузку операторов сравнения,
строковое представление.
4. Класс JSONSaver
Обеспечивает:

создание JSON-файла при отсутствии,
добавление записей,
удаление записей,
защиту от дубликатов по icao24.
5. Функции в utils.py
Содержат:

фильтрацию по стране,
фильтрацию по высоте,
фильтрацию самолетов в воздухе,
сортировку по высоте,
сортировку по скорости,
вывод данных в консоль.
